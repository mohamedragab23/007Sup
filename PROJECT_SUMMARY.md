# ملخص المشروع - نظام إدارة المشرفين

## نظرة عامة

تم إنشاء نظام إدارة شامل للمشرفين والمناديب باستخدام Next.js 14 مع TypeScript، مع تكامل كامل مع Google Sheets API. النظام يدعم واجهة عربية حديثة مع دعم RTL كامل.

## التقنيات المستخدمة

- **Next.js 14**: إطار عمل React مع App Router
- **TypeScript**: للكتابة الآمنة
- **Tailwind CSS**: للتصميم
- **Google Sheets API**: لتخزين البيانات
- **Recharts**: للرسوم البيانية
- **JWT**: للمصادقة
- **Vercel**: للنشر

## الأوراق المطلوبة في Google Sheets

1. **المشرفين**: بيانات المشرفين
2. **المناديب**: بيانات المناديب
3. **البيانات اليومية**: بيانات يومية لكل مندوب
4. **الخصومات**: خصومات المشرفين
5. **السلف**: سلف المشرفين
6. **استعلام أمني**: استعلامات أمنية
7. **المعدات**: تكلفة المعدات
8. **الأهداف**: أهداف وبونص المشرفين

## خطوات النشر

1. إعداد Google Cloud Project و Service Account
2. إعداد متغيرات البيئة
3. تثبيت الحزم: `npm install`
4. تشغيل محلي: `npm run dev`
5. النشر على Vercel: اربط المستودع وأضف متغيرات البيئة
